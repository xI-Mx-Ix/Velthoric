# Planned README:

Дополнение для ViveCraft, который добавляет в игру поддержки физических взаимодействий с предметами.

## Структура проекта

> [!WARNING]
> На данный момент, проект является прототипом, а я неумелым разработчиком. Вследствие этого код **будет** переписываться, а его логика меняться.
> 
> Касательно использования Velthoric целиком, а не как библиотеки: мне хотелось разобраться в устройстве его работы «под капотом», попробовать добавить поддержку минорных версий (в будущем), а так же я планирую вырезать/добавлять некоторый функционал.
> 
> Возможно, позднее я перейду к использованию Velthoric как библиотеки.

Будет описано позже.
Пока что проект представляет собой сборную солянку из исходников различных модов и шаблонов и требует более детального разбор и очистки.
